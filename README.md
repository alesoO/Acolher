## 🐾 Sistema de Adoção de Animais

Este projeto tem como objetivo desenvolver um sistema web para facilitar o processo de adoção de animais, conectando instituições, protetores independentes e possíveis adotantes de forma simples e eficiente.

### 📌 Funcionalidades

- Cadastro de animais disponíveis para adoção
- Exibição de informações detalhadas (espécie, idade, porte, fotos)
- Cadastro de adotantes
- Solicitação de adoção
- Controle de status dos animais (disponível, em processo, adotado)

### 🎯 Objetivo

O sistema busca melhorar a organização e a transparência no processo de adoção, reduzindo a burocracia e aumentando as chances de encontrar lares adequados para os animais.

### 🚀 Benefícios

- Facilidade na conexão entre adotantes e responsáveis pelos animais
- Centralização das informações
- Acompanhamento do processo de adoção em tempo real
- Maior visibilidade para animais disponíveis
